I Don't Wanna Taco 'Bout It uses the Dandelion API for Semantic Text Analysis of the user's responses and the Zomato API to provide restaurant recommendations. It considers the user's cumulative sentiment score to present the user with recommendations of Mexican restaurants within a specific price range in a specific locality.
